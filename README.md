## vue-upload-folder

> 使用方法

```
    <vue-gic-space
      id="dropzone"
      ref="myDropzone"
      :options="dropzoneOptions"
      :action="folderUrl"
      :parentId="parentId"
      :useCustomSlot="true">
      <i class="el-icon-upload"></i>
      <div class="el-upload__text">将图片或文件夹拖到此处上传，或点击<b>上传</b>图片</div>

      <div>最大不能超过30张图片</div>
    </vue-gic-space>
```
